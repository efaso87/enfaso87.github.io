Love Catcher

This is an gaming application created using Python (PyGame) that allows players to control a "cupid" character and catch as many hearts as possible, unlocking achievements and leveling up as they play. The game tracks progress, with players aiming to catch hearts to reach new levels and break high scores. It gets progressively harder as players advance, and the game ends when they miss five hearts.
